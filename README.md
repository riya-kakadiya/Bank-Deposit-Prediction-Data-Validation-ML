# Bank Deposit Subscription Prediction

## Project Overview
This project focuses on building a data-driven prediction system to identify whether a customer will subscribe to a bank deposit. The goal is to ensure strong data quality, validation, and analytical accuracy, similar to real-world enterprise data operations.

The project demonstrates end-to-end data handling including data cleaning, validation, exploratory data analysis, feature engineering, and machine learning model evaluation.

---

## Key Objectives
- Perform data validation and quality checks  
- Analyze customer data using EDA  
- Build and evaluate multiple machine learning models  
- Compare model performance using evaluation metrics  
- Ensure reliable and consistent analytical results  

---

## Tools & Technologies
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  

---

## Machine Learning Models Used
- Logistic Regression  
- Decision Tree  
- Random Forest  
- Support Vector Machine (SVM)  
- Naive Bayes  

---

## Model Evaluation
Models were evaluated using:
- Accuracy score  
- Confusion matrix  
- Classification report  

Performance comparison was used to identify the most reliable model.

- `*.ipynb`  Data analysis, validation, and model building
